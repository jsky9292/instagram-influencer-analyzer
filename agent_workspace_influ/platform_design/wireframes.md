## 4. 화면 설계 (텍스트 기반 와이어프레임)

### 4.1. 메인 페이지 (로그인 전)
```
+----------------------------------------------------+
| [로고] [서비스소개] [요금제] [로그인] [회원가입]   |
+----------------------------------------------------+
|                                                    |
|  "AI로 당신의 브랜드를 빛내 줄 최적의 인플루언서를  |
|               지금 바로 찾아보세요"                  |
|  [ 캠페인 등록하기 (브랜드) ] [ 인플루언서 지원하기 ] |
|                                                    |
+----------------------------------------------------+
|              [ 주요 파트너사 로고 ]                |
+----------------------------------------------------+
|                                                    |
|  [플랫폼 핵심 기능 소개 섹션: AI매칭, 성과분석 등] |
|                                                    |
+----------------------------------------------------+
|          [ 실제 성공 캠페인 사례 ]                 |
+----------------------------------------------------+
|          [ 사용자 유형별 후기 ]                    |
+----------------------------------------------------+
| [회사정보] [이용약관] [개인정보처리방침] [고객센터] |
+----------------------------------------------------+
```

### 4.2. 사용자별 대시보드

#### 4.2.1. 인플루언서 대시보드
```
+----------------------------------------------------+
| [로고] [캠페인찾기] [내활동] [정산관리] [프로필]   |
+----------------------------------------------------+
| [OOO님, 환영합니다!] [알림] [채팅]                 |
+----------------------------------------------------+
| [진행중인 캠페인: 3건] | [예상 수입: ₩1,500,000]   |
+----------------------------------------------------+
|                                                    |
|  [ 진행 중인 캠페인 목록 (카드 형태) ]             |
|  - 브랜드명, 캠페인명, 진행상태, 마감일            |
|                                                    |
+----------------------------------------------------+
|  [ 나에게 온 새로운 제안 (3) ]                     |
+----------------------------------------------------+
|  [ AI 추천 맞춤 캠페인 ]                           |
|  - 캠페인 카드 (스와이프 형태)                     |
+----------------------------------------------------+
```

#### 4.2.2. 브랜드 대시보드
```
+----------------------------------------------------+
| [로고] [캠페인관리] [인플루언서찾기] [리포트] [결제] |
+----------------------------------------------------+
| [OOO님, 환영합니다!] [알림] [채팅]                 |
+----------------------------------------------------+
| [진행중 캠페인: 5건] | [총괄 KPI: ROI 250%]        |
| [총예산: ₩5,000,000] | [총도달: 1,234,567]         |
+----------------------------------------------------+
|                                                    |
|  [ 캠페인별 성과 요약 (그래프) ]                   |
|                                                    |
+----------------------------------------------------+
|  [ 진행 중인 캠페인 목록 (테이블 형태) ]           |
|  - 캠페인명, 기간, 예산, 상태, 주요성과            |
+----------------------------------------------------+
|  [ To-Do List ]                                    |
|  - [콘텐츠 검수 요청 (2)] [계약대기 (1)]           |
+----------------------------------------------------+
```

### 4.3. AI 매칭 화면 (브랜드용)
```
+----------------------------------------------------+
| [캠페인 선택: OOO 신제품 홍보]                     |
+----------------------------------------------------+
| [ AI 추천 인플루언서 ] [ 직접 검색 ] [ 지원자 목록 ] |
+----------------------------------------------------+
|                                                    |
|  [ 추천 인플루언서 목록 (카드 형태) ]              |
|  +-----------------------------------------------+   |
|  | [프로필사진] [이름/채널명] [매칭점수: 95점]   |   |
|  | [카테고리] [팔로워] [예상CPE] [예상ROI]       |   |
|  | [대표 콘텐츠 이미지 3개]                      |   |
|  | [ 상세 보기 ] [ 제안 보내기 ]                 |   |
|  +-----------------------------------------------+   |
|  ...                                               |
|                                                    |
+----------------------------------------------------+
| [필터] [정렬]                                      |
| - 카테고리, 팔로워 수, 예산, SNS채널 등            |
+----------------------------------------------------+
```

### 4.4. 채팅 및 계약 화면
```
+----------------------------------------------------+
| [채팅 목록] | [대화창]                               |
+----------------------------------------------------+
| [브랜드A]   | [인플루언서 OOO님과의 대화]            |
| [브랜드B]   | ...                                    |
| [인플루언서C] | 안녕하세요, 제안주신 캠페인 관련...    |
|             | [파일첨부] [이모티콘] [전송]           |
|             +--------------------------------------+
|             | [계약 조건 확인 및 수정]               |
|             | - 캠페인 기간: YYYY.MM.DD-MM.DD        |
|             | - 제공 금액: ₩500,000                  |
|             | - 콘텐츠 가이드라인.pdf                |
|             | [ 계약서 생성 ] [ 수정 요청 ]          |
+----------------------------------------------------+
```
